How it works

  1. Plan — LLM infers entity type, schema columns, and typed retrieval facets
  2. Search & Scrape — Multi-angle web search via Brave, async scraping with caching
  3. Rerank — Cross-encoder selects the most query-relevant pages for extraction
  4. Extract — LLM extracts structured entities with per-cell evidence and confidence
  5. Merge & Verify — Fuzzy dedup, cell-level verification, source-quality scoring, gap-fill
  6. Rank — 6-component weighted ranker: completeness, confidence, source quality, diversity, and more
